Saving a Template

  1. In Word menu at top click
    File >> Save As
  2. In Format dropdown menu of popup window choose "Word Template (.dotx)"
  3. In Save As textbox type "My_APA_Template" - for APA format
    or "My_MLA_Template - for MLA format
  4. Click "Save"
