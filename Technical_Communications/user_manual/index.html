What's this manual for?

Implementing MS Word formatting features to meet APA/MLA standards can be confusing and time consuming. Students devote much of their time in college writing academic papers. This internet based user manual was written in an effort to save student's time and make formatting easier. It includes simple step-by-step instructions for using MS Word 2011 software to create a basic and reusable customized APA/MLA template.

This manual was compiled and created by Diana Collins.
